# 贡献指南

## 硬件基线

本书的正文与示例统一使用 STM32F103ZET6：

- STM32F10X_HD
- startup_stm32f10x_hd.s
- Flash 512KB、SRAM 64KB

请不要在正文示例中混入其他容量、启动文件或引脚假设；若要支持其他板卡，应单独建立迁移说明。

## 提交教材内容时

每个实验章节应包含：

1. 学习目标与前置知识；
2. 接线或硬件前提；
3. 最小可运行步骤；
4. 预期现象；
5. 常见失败与排查；
6. 本章要点与下一章链接。

## 提交示例代码时

- 不提交 build 产物或真实密钥；
- 在 README 写明“已编译”“已烧录”或“仅骨架”；
- 说明目标板、工具链版本和接线；
- 优先提供最小可复现工程；
- 任何涉及电机、锂电、继电器或门锁的示例必须标明安全边界。
