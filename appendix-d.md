# 附录 D · 推荐资源（SPL版）

## 📖 必读文档

| 文档 | 说明 |
|------|------|
| **RM0008** — STM32F103 参考手册 | 所有寄存器描述的「圣经」。SPL 函数底层操作的就是这些寄存器 |
| **STM32F103xC/xD/xE Datasheet** | ZET6 电气特性、引脚表、封装 |

## 📖 书籍

| 书名 | 说明 |
|------|------|
| **The Definitive Guide to ARM Cortex-M3** (Joseph Yiu) | Cortex-M3 内核权威指南 |
| **Mastering STM32** (Carmine Noviello) | HAL/LL/CubeMX 为主，但 FreeRTOS 和架构部分通用 |
| **Making Embedded Systems** (Elecia White) | 嵌入式软件工程：状态机、架构、团队协作 |
