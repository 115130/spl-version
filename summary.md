## 核心能力清单

学完本书，你应该具备以下能力：

| 能力 | 达到的水平 |
|------|-----------|
| 阅读芯片数据手册（RM0008） | 能独立找到外设寄存器描述和地址 |
| 使用 GCC + Makefile + OpenOCD | 编译、烧录、命令行调试 |
| SPL GPIO 控制 | 输入/输出/中断/状态机，SPL 函数级和寄存器级都会 |
| 串口通信 | 轮询/中断/DMA 三种方式，环形缓冲区，AT 指令 |
| 定时器 | PWM、输入捕获、定时中断 |
| ADC | 单通道/多通道扫描 + DMA |
| I2C/SPI 总线 | SPL 驱动传感器/OLED/Flash/SD 卡 |
| DMA | 外设→内存/内存→外设，乒乓缓冲 |
| 文件系统 | FatFs 移植、读写 CSV、文件追加 |
| FreeRTOS | Task 创建/优先级/Queue/Semaphore/Mutex |
| 无线通信 | DX-WF24/ESP8266 AT 指令 WiFi、HC-05/HM-10 蓝牙 |
| MQTT | Broker 连接、发布/订阅、QoS、云端对接 |
| HTTP | GET/POST 请求、JSON 解析 |
| 系统集成 | 从传感器 → MCU → 无线模块 → 云平台 → App 的全链路 |
