## 17.1 三句话理解无线通信

1. **2.4GHz 是 IoT 甜点**——免费、全球通用、芯片便宜。WiFi、BLE、ZigBee 都在这。
2. **WiFi 高带宽高功耗**（适合上云），**BLE 低功耗**（适合手机直连）。
3. **AT 指令 = MCU 通过 UART 发字符串控制模块**。核心就三条：
   - `AT` → 模块回 `OK`：通信正常
   - `AT+CWJAP="SSID","PWD"` → 连 WiFi
   - `AT+CIPSEND=N` → 发 N 字节

## 17.2 AT 指令的本质

```
STM32                   无线模块 (DX-WF24)
 │                          │
 │── UART TX ──→ "AT\r\n" ─→│
 │←── UART RX ←── "OK\r\n"──│
 │                          │
 └── 这就是全部 —— 发字符串，等字符串 ──┘
```

AT 指令的收发封装（`UART2_Init`、`AT_SendCmd`、`AT_WaitResponse`）在第 18 章 §18.8 的 `main.c` 中。
