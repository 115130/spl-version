# 附录 A · STM32F103ZET6、供电与接线速查

> 本书唯一硬件基线：STM32F103ZET6 开发板。请把本附录当作每次接线前的检查表，而不是只在出错后才翻出来。板卡上已有器件的实际连线请填入 [板卡资源约定](./board-zet6-profile.md)，不要从芯片型号推断。

## A.1 固定工程参数

| 项目 | 本书配置 |
|---|---|
| 目标芯片 | STM32F103ZET6 |
| 内核 | Cortex-M3 |
| 最大系统时钟 | 72 MHz |
| Flash | 512KB |
| SRAM | 64KB |
| SPL 密度宏 | STM32F10X_HD |
| 启动文件 | startup_stm32f10x_hd.s |
| 链接脚本 | Flash 512KB、SRAM 64KB |
| 调试接口 | SWD + ST-Link |
| 模拟能力 | 3 个 ADC、2 个 DAC；模拟输入/输出的范围和负载必须另行核对 |

本书不以 C8T6、Medium Density 或其他容量的链接脚本为主线。遇到其他教程的配置时，先确认它是否适用于 ZET6。

## A.2 本书常用引脚

这些是教材的外接实验默认映射或 MCU 固定复用关系。开发板的板载外设可能使用不同引脚；接线前仍需看丝印、原理图和数据手册。一个引脚一次只能服务一个实验，冲突见 [板卡资源约定](./board-zet6-profile.md)。

| 功能 | 常用引脚 | 章节 | 注意事项 |
|---|---|---|---|
| LED | `board.h` 中定义；PC13 仅为常见默认 | 0、3、7 | 有效电平也必须由板卡确认 |
| USART1 外接调试 | PA9 / PA10 | 8 以后 | TX/RX 要与 USB-TTL 交叉；板载 USB 串口未必接此处 |
| USART2 模块通信 | PA2 / PA3 | 18、21–24 | WiFi/BLE 模块需共地；与具体模块实验前确认 |
| I2C1 | PB6 / PB7 | 10、25 | SDA/SCL 需要上拉 |
| SPI1 | PA5 / PA6 / PA7 | 11、13、25 | SCK/MISO/MOSI；每个从设备独立 CS，CS 不默认假设为 PA4 |
| ADC1 外接电位器 | PA0 / ADC1_IN0 | 9、25 | 输入只在 0–VDDA 内；与 PA0 按键/捕获实验互斥 |
| DAC1 / DAC2 | PA4 / PA5 | 9 | 输出不是任意电源；需考虑缓冲、负载和量程 |
| SWD | SWDIO / SWCLK / GND | 0、28 | 按板卡丝印接 ST-Link |

## A.3 上电和电平规则

- ZET6 的普通 GPIO 以 3.3V 逻辑为基线；不要把未知的 5V 数字输出直接接入 GPIO。
- 两个模块要进行 UART、I2C、SPI、RS485 通信时，先确认是否需要共地。
- 外设模块的“供电电压”与“IO 电平”是两件事；模块可由 5V 供电，不代表 TX/RX 一定是 3.3V。
- 模拟输入不能高于 VDDA，也不能低于 VSSA；DAC 的实际可达高低端还会受输出缓冲、负载和数据手册限制。
- 不要同时用 USB、ST-Link 3.3V、外接电源向同一板子随意供电。
- 接线错误、焦味、异常发热时，先断电；不要反复带电试错。

## A.4 常用接线模式

### UART

~~~text
MCU TX  ───→ 模块 RX
MCU RX  ←─── 模块 TX
MCU GND ───── 模块 GND
~~~

### I2C

~~~text
MCU SCL ───── 设备 SCL
MCU SDA ───── 设备 SDA
GND     ───── GND
3.3V    ───── VCC（以模块规格为准）
~~~

SDA/SCL 是开漏总线，通常需要上拉电阻；没有上拉时，设备可能完全没有响应。

### SPI

~~~text
MCU SCK  ───── 所有设备 SCK
MCU MOSI ───── 所有设备 MOSI
MCU MISO ←──── 所有设备 MISO
MCU CS   ───── 每个设备各自一个 CS
GND      ───── GND
~~~

## A.5 接线前最后检查

- [ ] 板子确实是 STM32F103ZET6；
- [ ] 工程使用 HD 启动文件与 512KB Flash 链接脚本；
- [ ] 电源、电平与 GND 已确认；
- [ ] TX/RX 已交叉；
- [ ] I2C 已确认上拉；
- [ ] 外部模块的峰值电流没有超过开发板供电能力；
- [ ] 所有线按引脚标签连接，不按颜色猜测。
- [ ] 本次实验没有和已有按键、ADC、捕获、PWM、SWD 或总线实验争用同一引脚/定时器。
