# 普中科技 玄武 STM32F103 开发板 —— 硬件规格

> **重要提示**：本文件根据普中官方配套文档整理，所有标注「标注」的器件来自文档目录确认存在，具体型号和引脚分配以你手中板子的实物和原理图为准。

---

## 1. 主控 MCU

| 项目 | 规格 |
|------|------|
| 型号 | **STM32F103VET6** 或 **ZET6** |
| 架构 | ARM Cortex-M3 |
| 主频 | 72MHz |
| Flash | 512KB（VET6/ZET6） |
| SRAM | 64KB（VET6/ZET6） |
| 调试接口 | SWD（可通过 ST-Link / DAP-Link 调试） |

> 部分版本可能用 STM32F103VET6（LQFP100 封装），部分用 ZET6（LQFP144 封装）。VET6 少一些引脚，但大部分外设兼容。

---

## 2. 板载存储器

| 器件 | 型号 | 容量 | 接口 | 用途 |
|------|------|------|------|------|
| SPI NOR Flash | **W25Q64** | 64Mbit / 8MB | SPI | 字库、图片、OTA 固件包 |
| SPI NOR Flash | **W25Q128** | 128Mbit / 16MB | SPI | 更大存储版本（可能有或无） |
| SPI NOR Flash | **W25X16** | 16Mbit / 2MB | SPI | 较老的 SPI Flash |
| I²C EEPROM | **AT24C02** | 2Kbit / 256B | I²C（地址 0x50） | 掉电保存配置参数 |
| 外部 SRAM | **IS62WV51216** | 512K×16bit = **1MB** | FSMC（并口） | LCD 帧缓冲、大内存计算 |

> 从文档来看 **W25Q64 和 AT24C02 是必有的**，W25Q128 和 W25X16 可能是不同批次或选配。

---

## 3. 板载传感器

| 器件 | 类型 | 接口 | 说明 |
|------|------|------|------|
| **DHT11** | 温湿度 | 单总线（1-Wire） | 精度 ±2°C / ±5% RH |
| **DS18B20**（DS1820） | 数字温度 | 单总线（1-Wire） | 精度 ±0.5°C，支持级联 |
| **MPU6050** | 六轴运动 | I²C | 三轴加速度计 + 三轴陀螺仪 + 硬件 DMP 姿态解算 |
| **MMA8653FCR1** | 三轴加速度计 | I²C | 低功耗加速度传感器 |
| **HMC6352** | 电子罗盘 | I²C | 磁力计 / 方向传感器 |
| **LM77** | 数字温度 | I²C | 带过温报警输出的温度传感器 |

---

## 4. 通信接口

| 器件 | 类型 | 说明 |
|------|------|------|
| **CH340** | USB → UART | USB 转串口，连接到板子上的某个 UART，供串口终端/printf 调试 |
| **SP3232** | UART → RS232 | RS232 电平转换，连接传统串口设备 |
| **SP3485** | UART → RS485 | RS485 差分通信，工业现场总线 |
| **TJA1050** | CAN 收发器 | CAN 总线接口，接 STM32 的 CAN 控制器引脚 |

---

## 5. 显示与触摸

| 器件 | 类型 | 接口 |
|------|------|------|
| **XPT2046** (ET2046) | 四线电阻式触摸屏控制器 | SPI |
| **ADP8860** | LED 背光驱动器 | I²C（带电荷泵，可调亮度） |

---

## 6. 时钟与电源

| 器件 | 类型 | 接口 |
|------|------|------|
| **MCP79412** | RTC 实时时钟 | I²C（带 SRAM 和 EEPROM） |
| **AMS1117** | 稳压器 | LDO 3.3V 输出 |

---

## 7. 协议与外设对照表

| 协议 | 板载器件 |
|------|----------|
| **I²C** | AT24C02、MPU6050、MMA8653、HMC6352、ADP8860、MCP79412、LM77 |
| **SPI** | W25Q64、W25Q128、W25X16、XPT2046 |
| **FSMC（并口）** | IS62WV51216 SRAM（也可能接 LCD） |
| **单总线（1-Wire）** | DHT11、DS18B20 |
| **UART** | CH340（USB 调试口）、SP3232（RS232）、SP3485（RS485） |
| **CAN** | TJA1050 |

---

## 8. 调试与下载

| 方式 | 接口 | 说明 |
|------|------|------|
| **SWD** | 板载 SWD 接口（4-5pin） | 用 ST-Link / DAP-Link / J-Link 调试和烧录 |
| **USB→UART** | CH340 的 USB 口 | 通过串口 ISP 下载（需配合 Bootloader 模式） |
| **板载调试器** | 可能带 | 你的板子不确定是否焊接了调试器芯片 |

---

## 9. 这本书中会用到的外设（按章节）

| 章节 | 外设 | 板载对应器件 |
|------|------|-------------|
| 03 GPIO | LED + 按键 | 板载 LED / 按键 |
| 07 定时器 | PWM | LED 或 ADP8860 背光控制 |
| 08 UART | 串口通信 | CH340（调试串口）/ SP3232 / SP3485 |
| 09 ADC | 模数转换 | 板载电位器 |
| 10 I²C | I²C 总线 | AT24C02 / MPU6050 / OLED 模块 |
| 11 SPI | SPI 总线 | W25Q64 / XPT2046 |
| 12 DMA | DMA 控制器 | ADC + 多通道，SPI Flash |
| 13 存储器 | FatFs | W25Q64 + SPI |
| 17-20 无线 | WiFi/BLE | 外接 DX-WF24 / ESP8266 |
| 21-24 云 | MQTT | 同上 |
| 25-27 项目 | 综合 | 全部 |
| 28-30 产品化 | 低功耗/调试 | 全部 |

---

> **来源**：普中科技官方配套文档，2024 年版本。标记为 W25Q64 + AT24C02 + IS62WV51216 SRAM + MPU6050 等全套传感器方案。具体引脚号和跳线配置请查阅你手头板子的原理图 PDF。
